Shaapes - A Javascript Background Generator

by Josh Drumm (@wwnjp)

This is a small web application for the purpose of generating images of repeating geometric shapes in various color palettes. This was specifically designed so I could create minimal hi-res background images. This is an entirely Javascript/jQuery-based application with no backend processor at all. This app uses canvas to generate all of the images.

Once you tweak the parameters and find an image that you like, you can click the "Download" button, which will let you save it to your device.

Special Thanks

This project uses the jQuery Javascript library.

The icons are from Typicons

It also uses the public API provided by ColourLovers.com, who store a collection of amazing colo(u)r palettes.
